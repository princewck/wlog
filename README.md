## 一款定制化博客系统

### 可能会用到的技术栈：

#### 前端（包括admin管理界面）:
- ReactJs, Redux, 
- Angular, RxJs or Mobx

#### 后端：
- NodeJs(Koa框架)
- MongoDB
- Elasticsearch

#### 功能计划
- 日志编辑
  - 富文本
  - markdown
  - 自动保存
  - 图片本地化
  - 代码高亮
  - 个性化内容模板
- 日志列表
  - 模糊搜索
  - 标签体系
  - 时间段分组
  - 可视化统计信息
  - 分享
- 适配：响应式布局
  - 移动端
  - PC
- 定制化：
  - 导航可配置
  - 主题可配置
  - 个人展示，作品展示
